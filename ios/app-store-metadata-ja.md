## Description Draft

爺コブシは、演歌のコブシをモチーフにしたリズムアクションゲームです。

主人公は、孫を救うために立ち上がった立石小次郎。商店街、倉庫、道場、山道、改造ガレージ、赤門、そして秘密結社の本拠地へ。流れてくる合図に合わせて叩き、長押しし、連打して、敵の体力を削っていきます。

主な特徴:

- タップ、長押し、連打で遊ぶリズムバトル
- 7ステージ構成の短編ストーリー
- イージー、ノーマル、ハードの難易度選択
- 日本語と英語の表示切り替え
- 入力補正、セーブ、一時停止に対応
- 横画面で遊びやすいシンプル操作

派手すぎず、でも熱い。演歌の節回しに合わせて、立石小次郎の一撃を決めてください。

## App Review Notes Draft

This app is a self-contained rhythm action game. It does not require an account, external network service, ads, or in-app purchases in the current prototype.

Gameplay:

1. Start the game from the title screen.
2. Select a difficulty.
3. Tap the rhythm input area when notes reach the judgment line.
4. Hold during HOLD notes and release at the release marker.
5. Mash during red MASH windows.

The app is intended for landscape play.
