基本

  • TAP：ノーツが金の判定線に重なった瞬間にタップ。
  • HOLD：金の「長押」で押し続け、白い「離す」が判定線に来たら離す。
  • 連打：赤い枠が出ているあいだ、目標回数まで連打。多すぎるとランクが落ちる。失敗しても体力は減らない。
  • 細かいTAP：同じレーンで短く続くノーツはすべてタップ。長押しではない。
  • 連撃：5連、10連で追撃。短いTAPをつなぐほど敵体力を大きく削れる。

便利機能

  • 入力補正：音や画面の遅れに合わせて ms でずらせます。
  • 表示 大きめ：タイミング読み（次の拍までの ms）と文字を大きくします。
  • セーブ：プレイ中は画面右上の「セーブ」で周回別に保存できます。
  • 一時停止：戦闘中に画面右上の「一時停止」または Esc。

Basics

  • TAP when the note meets the gold judgment line.
  • HOLD: hold on the gold long-hold marker, then release when the white release marker reaches the line.
  • MASH: while the marker is in the window, mash to the target count—wild overspam lowers rank. Mash misses do not cost HP.
  • Dense TAP: quick taps on the same lane are still taps—release between each one, not a hold.
  • Combo strikes: 5- and 10-hit chains add pursuit damage, so dense taps matter.

Comfort

  • Input offset (ms) fixes audio/video lag.
  • Comfort / larger: shows “ms to next beat” and bumps type size.
  • Save: use the top-right Save button during a run. Saves are split by first loop and loop 2+.
  • Pause: top-right Pause or Esc during battle.

スマホでは横向き推奨。判定ラインとノーツが見やすくなります。
